# Kalkulator Zakat Penghasilan

Aplikasi web sederhana untuk menghitung zakat penghasilan bulanan sesuai prinsip syariah, dibuat sebagai bagian dari proses belajar web development.

## Fitur
- Input penghasilan bulanan
- Otomatis cek apakah penghasilan sudah mencapai nisab (Rp7.000.000/bulan)
- Hitung zakat 2.5% jika sudah memenuhi nisab
- Format angka otomatis (Rp1.000.000, bukan Rp1000000)
- Bisa hitung dengan tombol atau tekan Enter

## Teknologi yang Digunakan
- HTML
- CSS (Flexbox, Box Model)
- JavaScript (DOM Manipulation, Validasi Logika)
